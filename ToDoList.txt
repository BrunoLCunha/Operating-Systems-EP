- número médio de instruções executadas não pode passar do valor do quantum
- número médio de instruções é a soma das instruções executadas de cada um dividido pela soma dos quantuns que foram dados. Exemplo:
Q   1   1   1   2
N   3   3   2   3
    P1  P2  P3  P1
NMedioInstrucoes = (2 + 3 + 2 + 3 + 2)/6

- N médio de trocas. Exemplo:
Q   1   1   1   2   1   1
N   3   3   2   3   2   2
    P1  P2  P3  P1  P4  P1
NMedioTrocas = 6/4 = 1,5

-Independente de fazer entrada e saida quando o processo voltar ganha a quantidade de quantum que foi dada anteriormente + 1
